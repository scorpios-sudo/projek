⚡ Perhitungan Pips:
- Forex (EUR/USD, GBP/USD, dll): 1 pip = 0.0001 → Pips = |Exit - Entry| / 0.0001
- XAUUSD / GOLD: 1 pip = 0.01 → Pips = |Exit - Entry| / 0.01
Profit = Pips × Lot × 100 (karena 0.01 lot = $1/pip)
Net Profit = Profit - Komisi - Swap
📌 Contoh XAUUSD: Entry 4.57794, Exit 4.51684, Lot 0.13, Arah Sell → Selisih = 0.0611, Pips = 0.0611 / 0.01 = 611, Profit = 611 × 0.13 × 100 = $7,943.00 ✅
