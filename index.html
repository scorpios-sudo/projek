🖥️ Cuplikan Kode Inti

import MetaTrader5 as mt5
import pandas as pd
import numpy as np

# Inisialisasi & login
mt5.initialize()
mt5.login(12345678, password="pass", server="Exness-MT5Trial")

def get_ma_cross(symbol):
    rates = mt5.copy_rates_from_pos(symbol, mt5.TIMEFRAME_M5, 0, 100)
    df = pd.DataFrame(rates)
    df['ma_fast'] = df['close'].rolling(5).mean()
    df['ma_slow'] = df['close'].rolling(20).mean()
    return 1 if df['ma_fast'].iloc[-1] > df['ma_slow'].iloc[-1] else -1

# Contoh order BUY
if get_ma_cross("EURUSD") == 1:
    request = {
        "action": mt5.TRADE_ACTION_DEAL,
        "symbol": "EURUSD",
        "volume": 0.1,
        "type": mt5.ORDER_TYPE_BUY,
        "price": mt5.symbol_info_tick("EURUSD").ask,
        "sl": price - 200 * point,
        "tp": price + 300 * point,
        "deviation": 20,
        "magic": 234000,
        "comment": "Python MA bot",
        "type_time": mt5.ORDER_TIME_GTC,
        "type_filling": mt5.ORDER_FILLING_IOC,
    }
    mt5.order_send(request)

🔧 Cara Setup di Lokal / VPS

  1. Install Python 3.7+ dan pustaka MetaTrader5 pandas numpy.
  2. Download & install MetaTrader 5 dari Exness (atau broker lain).
  3. Login ke akun Exness (demo atau real) di MT5, catat login, password, dan server.
  4. Clone repositori atau salin kode bot.
  5. Ubah kredensial di config.py (jangan di-commit ke publik!).
  6. Jalankan bot dengan python bot_simple.py. Untuk 24/7, deploy ke Windows VPS.
